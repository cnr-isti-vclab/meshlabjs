meshlabjs
=========

A javascript client based mesh processing tool. Built using [vcg library](http://vcg.sourceforge.net), emscripten and webgl. Developed by the [Visual Computing Lab](vcg.isti.cnr.it) of [ISTI - CNR](www.isti.cnr.it).
